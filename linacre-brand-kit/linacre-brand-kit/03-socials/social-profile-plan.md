# Social Profile Plan — Linacre.site

## 1. Profile Goal

Your social profiles should make people understand three things immediately:

1. You are a full-stack and AI engineer.
2. Linacre.site is useful: tools, roadmap, AI Lab, changelog.
3. The brand has a recognisable amber retro-modern signature.

## 2. Universal Profile Setup

### Avatar
Use the amber hex-pulse mark on a dark background.

### Display Name
Recommended:

- David Linacre
- David Linacre · Linacre.site
- Linacre.site

### Handle
Use consistent handles where possible:

- `@LIN4CRE`
- `@linacre_site`
- `@davidlinacre`

### Link
Primary link:

- `https://linacre.site`

### Short Bio Formula

```text
[Role] building [project] — [value].
```

Example:

> Full-stack + AI engineer building Linacre.site — tools, roadmaps, experiments, and calm web systems.

## 5. Posting Rhythm

Simple weekly rhythm:

- Monday: tool recommendation
- Tuesday: build note
- Wednesday: quick tip or code snippet
- Thursday: AI Lab or prompt experiment
- Friday: changelog/update
- Weekend: longer reflection, roadmap, or project walkthrough

## 6. Visual Consistency

Every social post should use:

- dark/amber palette,
- strong title,
- one central idea,
- same logo placement,
- consistent margins,
- calm technical tone.

Do not over-design each post. Consistency beats complexity.
