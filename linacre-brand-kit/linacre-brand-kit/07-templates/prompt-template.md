# Prompt Template — Linacre.site

Use this whenever creating a new AI prompt.

```text
You are creating [asset type] for Linacre.site, a full-stack and AI engineering portfolio.

Brand identity:
- amber
- hexagon
- pulse
- cyber
- soft retro-modern
- calm premium developer workspace

Asset purpose:
[Where this asset will be used]

Audience:
[Self-taught developers / recruiters / collaborators / social followers]

Required content:
[Text, logo, sections, duration, dimensions, etc.]

Visual/audio direction:
[Colours, mood, instruments, layout, typography, motion, etc.]

Style constraints:
[Readable, minimal, accessible, loopable, small-size safe, etc.]

Avoid:
[Generic SaaS, harsh neon, clutter, robots, blue AI clichés, etc.]

Output format:
[SVG / PNG / MP4 / WAV / Markdown / React component / CSS]
```

## Checklist Before Running A Prompt

- [ ] Did I include the asset purpose?
- [ ] Did I include size/duration?
- [ ] Did I include required text?
- [ ] Did I include the Linacre.site identity?
- [ ] Did I say what to avoid?
- [ ] Did I ask for export-ready output?
