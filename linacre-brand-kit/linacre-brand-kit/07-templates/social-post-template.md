# Social Post Template — Linacre.site

## One-Post Structure

```text
Hook:
[One clear sentence that gives people a reason to read]

Context:
[What you built, learned, tested, or found]

Value:
[Why this helps another builder]

Detail:
[One example, screenshot, mini-list, or code snippet]

CTA:
[Simple next step]
```

## Example

```text
Hook:
A portfolio is stronger when it behaves like a product.

Context:
I rebuilt Linacre.site around a toolkit directory, AI Lab, roadmap, and changelog.

Value:
It gives visitors useful things to explore instead of only reading a static about page.

Detail:
The changelog now shows each release like a small product update.

CTA:
What would you add to a developer portfolio?
```

## Carousel Structure

1. Cover: bold idea
2. Problem: why it matters
3. Principle: the main lesson
4. Example: show it in practice
5. Tools: useful stack/resources
6. Checklist: action steps
7. CTA: save/follow/visit

## Short Video Script Structure

```text
0–2s: Hook on screen
2–6s: Show the thing
6–11s: Explain the useful detail
11–15s: CTA + Linacre.site logo sting
```
