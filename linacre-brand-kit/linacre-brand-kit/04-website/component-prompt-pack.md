# Component Prompt Pack — Linacre.site

Use these smaller prompts when building individual components in Cursor, v0, Lovable, or any coding assistant.

---

## Hero Component

> Build a responsive React/TypeScript hero component for Linacre.site. Use deep ink background, amber radial glow, subtle hex grid, terminal-style eyebrow text, large Space Grotesk-style headline, readable subheading, two CTAs, and a small animated amber pulse element. Include accessible buttons, reduced-motion support, and clean mobile layout.
