# Launch Checklist — Linacre.site Brand

Use this before publishing the refreshed brand, website, or social profiles.

## Brand Basics

- [ ] Final logo mark selected
- [ ] Wordmark selected
- [ ] Favicon package exported
- [ ] Social avatar exported
- [ ] Open Graph image exported
- [ ] LinkedIn/X/GitHub banners exported
- [ ] Brand colours applied consistently
- [ ] Typography set and fallbacks defined
- [ ] Tone of voice checked

## Website

- [ ] Hero clearly explains who you are
- [ ] Toolkit is searchable/filterable
- [ ] AI Lab has trust/security notes
- [ ] Roadmap is easy to follow
- [ ] Changelog shows version history
- [ ] Footer has contact and links
- [ ] Open Graph metadata works
- [ ] Sitemap and robots are correct
- [ ] Favicon displays in browser tab
- [ ] Mobile layout checked
- [ ] Dark/light mode checked if applicable

## Accessibility

- [ ] Colour contrast is readable
- [ ] Keyboard focus states visible
- [ ] Search/filter usable by keyboard
- [ ] Images have alt text or are decorative
- [ ] Motion respects prefers-reduced-motion
- [ ] Buttons and links have accessible names
- [ ] Headings are semantic

## Performance

- [ ] Images compressed
- [ ] SVGs optimised
- [ ] No unnecessary large libraries
- [ ] Lighthouse checked
- [ ] Fonts optimised or safely loaded
- [ ] No layout shift from media

## Social Profiles

- [ ] LinkedIn avatar/banner updated
- [ ] X avatar/header updated
- [ ] GitHub README/banner updated
- [ ] YouTube/TikTok/Instagram avatar prepared
- [ ] Bios aligned across platforms
- [ ] Website link added everywhere
- [ ] Pinned launch post planned

## Audio / Video

- [ ] 3s logo sting exported
- [ ] 8s social intro exported
- [ ] 15s social cue exported
- [ ] 30s loop exported if using website audio
- [ ] Video logo animation exported
- [ ] Audio is not autoplaying on website

## Final Review

Ask:

- [ ] Is it useful?
- [ ] Is it calm?
- [ ] Is it technical?
- [ ] Is it warm?
- [ ] Is it recognisably Linacre.site?
