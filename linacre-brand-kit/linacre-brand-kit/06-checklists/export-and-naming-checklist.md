# Export and Naming Checklist — Linacre.site

## Naming Pattern

Use:

```text
linacre-[category]-[asset]-[size-or-duration]-v[number].[extension]
```

Examples:

```text
linacre-logo-hex-pulse-dark-v01.svg
linacre-logo-wordmark-horizontal-v01.svg
linacre-icon-favicon-32x32-v01.png
linacre-social-linkedin-banner-1584x396-v01.png
linacre-web-og-card-1200x630-v01.png
linacre-audio-sting-03s-v01.wav
linacre-video-logo-sting-03s-v01.mp4
```

## Website Exports

- [ ] Open Graph 1200x630
- [ ] Hero background SVG/PNG
- [ ] Decorative grid/pattern SVG
- [ ] Logo SVG imported into project
- [ ] Brand CSS tokens added
- [ ] Metadata references correct image paths

## Audio Exports

- [ ] WAV master 48kHz/24-bit
- [ ] MP3 320kbps for web/social
- [ ] Short tail cleaned
- [ ] Loop tested for seamless playback
- [ ] Loudness checked

## Versioning

- `v01` = first usable export
- `v02` = meaningful design change
- `v03` = final polish/revision
- Keep old versions until the brand is locked.
