# Logo and Icon Prompts — Linacre.site

Use these prompts for Midjourney, DALL·E, Firefly, Ideogram, Leonardo, Recraft, Figma AI, or a designer. For best results, paste the **Master Brand Prompt** first.

---

## 1. Main Logo Mark — Hex Pulse

### What this is for
The primary symbol for Linacre.site. Use it for favicon, social avatar, app icon, loading spinner, video intro, watermark, and sticker-style graphics.

### Prompt

> Create a minimal premium logo mark for “Linacre.site”, a full-stack and AI engineer portfolio. The mark should combine a clean hexagon outline, a subtle amber pulse line, and a small terminal/coding signal. Style: soft retro cyber, modern developer brand, calm and premium. Use deep ink black background, warm amber glow, subtle cream highlights, very restrained cyan accent only if needed. The logo must work at small favicon size, so keep it simple, geometric, readable, and distinctive. It should feel like amber terminal light inside a quiet engineering dashboard. Flat vector style, crisp edges, balanced negative space, no complex illustration, no mascot, no robot, no stock icon look.

### Settings / directions

- Aspect ratio: 1:1
- Style: vector, minimal, premium
- Background: transparent and dark versions
- Export: SVG, PNG 1024, PNG 512, favicon 32

### Negative prompt

> no robot, no gaming logo, no lightning bolt, no shield cliché, no blue corporate SaaS icon, no detailed circuit board, no tiny unreadable text, no 3D chrome, no excessive neon, no crypto coin style.

## 2. Wordmark — Linacre.site

### What this is for
Use on the website header, social banners, GitHub README, video title cards, and PDF/pitch materials.

### Prompt

> Design a clean wordmark for “Linacre.site”. Pair a minimal amber hex-pulse icon with a modern geometric wordmark. The word “Linacre” should feel confident and technical; “.site” should be smaller, softer, and treated like a domain suffix. Visual style: premium full-stack engineer portfolio, soft retro-modern cyber, warm amber on deep ink background. Use generous spacing, precise alignment, and subtle terminal/code personality. Make it readable at small sizes and elegant at large sizes. Include versions horizontal, stacked, dark background, transparent background, and monochrome.

### Export requirements

- `linacre-wordmark-horizontal.svg`
- `linacre-wordmark-stacked.svg`
- `linacre-wordmark-dark.png`
- `linacre-wordmark-light.png`
- `linacre-wordmark-mono.svg`

### Notes
The wordmark should not look like a metal band, crypto token, or esports team. It should look like a serious personal engineering brand.

## 3. Favicon

### What this is for
Browser tab, bookmarks, pinned tabs, mobile shortcuts.

### Prompt

> Create an ultra-simple favicon for Linacre.site based on an amber hexagon and a small pulse/cursor mark. It must be recognisable at 16x16 and 32x32 pixels. Use a dark ink background and one strong amber symbol. Avoid text unless it is a single simple “L”. The favicon should feel like a tiny glowing terminal signal.

### Export requirements

- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` 180x180
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

### Detail rule
If the icon has more than three visual ideas, it is too complex.

## 4. Social Avatar

### What this is for
LinkedIn, X, GitHub, YouTube, TikTok, Discord, profile cards.

### Prompt

> Create a social avatar for Linacre.site using the hex-pulse logo mark. It should sit centred inside a dark rounded-square or circular field with a soft amber radial glow. The image should be instantly recognisable at profile-photo size. Premium, calm, retro-modern, developer portfolio aesthetic. Add a very subtle grid or terminal texture in the background, but keep the mark clear. No face, no mascot, no busy code background.

### Export requirements

- 400x400 PNG
- 800x800 PNG
- SVG source if possible

### Platform note
For LinkedIn and GitHub, use the clearer version. For TikTok/YouTube, a slightly stronger glow is acceptable.

## 5. App Icon / PWA Icon

### What this is for
If Linacre.site is installed as a PWA or saved to mobile home screen.

### Prompt

> Create a PWA app icon for Linacre.site. Use a deep ink rounded-square base, a centred amber hexagon monogram, and a small glowing pulse line. The icon should feel like a premium developer tool app, not a corporate SaaS product. Keep it simple, high contrast, and readable on mobile. Add gentle inner glow and subtle depth, but avoid heavy 3D.

### Export requirements

- 192x192 PNG
- 512x512 PNG
- maskable icon safe-area version

## 6. Open Graph / Social Share Card

### What this is for
Preview image when the site is shared on LinkedIn, X, Discord, Slack, etc.

### Prompt

> Create an Open Graph share image for Linacre.site. Size 1200x630. Dark ink background with soft amber radial glow, subtle hexagon grid, and minimal terminal details. Left side: Linacre.site wordmark. Main headline: “Full-Stack & AI Engineering Toolkit”. Subtext: “Tools, roadmaps, experiments, and clean web systems.” Include a small amber pulse line or waveform. Premium, readable, calm, retro-modern cyber. Keep text large and correctly spelled.

### Export requirements

- 1200x630 PNG
- Keep safe margins of at least 80px
- Test on dark and light social feeds

## 7. Monochrome Logo Prompt

### What this is for
Documents, stamps, one-colour printing, favicons, masks.

### Prompt

> Convert the Linacre.site hex-pulse logo into a one-colour monochrome vector. Preserve the hexagon, pulse, and monogram/cursor idea using clean negative space. No gradients, no glow, no shadows. It must work in pure black, pure white, and single amber.

## 8. Designer Handoff Notes

When giving this to a human designer, say:

- The icon must pass the 16px test.
- The wordmark must work in a website header at 120–180px wide.
- Amber on dark is the primary use case.
- The brand is calm premium engineering, not loud cyberpunk.
- Provide SVG source, PNG exports, and favicon package.
