# Audio Brand Prompts — Linacre.site

Use these with music-generation tools such as Suno, Udio, Stable Audio, MusicFX, or a human producer. The Linacre.site audio identity is **quiet soft retro easy-listening with a modern digital twist**.

---

## 1. Master Audio Direction

### What this is for
Use this as the base description for any Linacre.site music.

### Prompt

> Create instrumental brand music for “Linacre.site”, a full-stack and AI engineer portfolio with an amber, hexagon, pulse, cyber identity. Style: quiet soft retro easy-listening with a modern digital twist. Use warm felt piano, vintage Rhodes chords, subtle 80s synth pads, soft lo-fi drums, gentle bass, tiny digital pulse accents, and clean modern production. Start with a quiet intimate piano intro, build into a fuller warm verse, then bloom into a tiny emotional chorus lift that feels big in the listener's head but stays minimal and tasteful. Mood: calm, intelligent, premium, nostalgic, focused, friendly, cyber-lounge, late-night coding energy. Tempo 82–90 BPM. No vocals. No harsh EDM. No heavy drums. No cheesy corporate music. Loop-friendly.

## 2. 3-Second Logo Sting

### What this is for
Logo animation, video end card, website button success sound if used very softly.

### Prompt

> Create a 3-second instrumental logo sting for Linacre.site. Start with one soft felt piano note, add a tiny amber digital pulse, then resolve into a warm Rhodes chord with a gentle sub bass bloom. Retro-modern, calm, premium, developer brand. It should feel like a terminal waking up in a quiet room. No vocals, no drums, no loud hit, no cinematic boom. End cleanly with a short tail.

### Export

- WAV 48kHz 24-bit master
- MP3 320kbps for web/social
- Filename: `linacre-audio-sting-03s-v01.wav`

## 7. Alternative Genre Mixes

### Softer lounge version
> Soft jazz lounge + lo-fi hip hop + ambient synth. Felt piano, Rhodes, brushed percussion, warm bass, subtle digital pulse. Calm amber developer brand, premium and intimate.

### More retro version
> 1980s soft synthwave + easy-listening piano. Warm analog pads, gentle arpeggio, Rhodes chords, low-volume lo-fi drums. Nostalgic but modern, no big drums.

### More technical version
> Minimal ambient techno influence at very low intensity. Tiny digital pulses, soft sub, glassy synth pad, sparse piano, no club beat. Calm AI lab energy.

> No vocals, no rap, no hard rock, no trap hats, no dubstep, no aggressive EDM, no cinematic boom, no corporate ukulele, no cheesy handclaps, no loud snare, no horror ambience, no extreme glitch, no jingle bells, no happy stock music.

## 9. Audio Usage Rules

- Use the 3s sting for logo moments.
- Use the 8s intro for branded videos.
- Use the 15s cue for short social clips.
- Use the 30s loop only when the user controls audio.
- Use the 60s bed for demos/walkthroughs.
- Keep final loudness comfortable: around -16 LUFS for web/social, quieter for background use.
