# Linacre.site Brand Kit

A drop-in brand folder for **Linacre.site**: a full-stack and AI engineering portfolio with a soft retro-modern identity.

This kit is designed to live inside your website/project folder. It gives you a practical brand system, asset folders, reusable AI prompts, export rules, social templates, website direction, and audio-brand guidance.

---

## Brand Snapshot

**Name:** Linacre.site  
**Personality:** calm, technical, clever, warm, premium, self-taught, builder-led  
**Core visual cues:** amber glow, hexagon, pulse, terminal, cyber-lounge, clean code, retro warmth  
**Core audio cues:** quiet piano, warm Rhodes, subtle synth pulse, lo-fi/easy-listening drums, small emotional lift  
**Positioning:** a personal full-stack and AI engineering hub for tools, builds, learning routes, and experiments.

**Short brand sentence:**  
> Linacre.site is a calm, amber-coded engineering hub for full-stack craft, AI tools, and self-taught builders.

---

## How To Use This Folder

1. Drop `linacre-brand-kit/` into your website repository or creative project folder.
2. Read `00-brand-system/brand-brief.md` first.
3. Copy the prompt from `01-prompts/master-brand-prompt.md` into any AI tool before using the specialist prompts.
4. Generate assets into `02-assets/`.
5. Use `03-socials/` for social profile setup and content planning.
6. Use `04-website/` for homepage, UI, component, and metadata direction.
7. Use `05-audio-brand/` for the website/social audio identity.
8. Use the checklists in `06-checklists/` before publishing.

## Best Build Order

### Phase 1 — Lock the identity
- Pick the final logo direction from `logo-and-icon-prompts.md`.
- Generate the main logo, wordmark, favicon, and social avatar.
- Save exports into `02-assets/logo/` and `02-assets/svg/`.

### Phase 2 — Update the website
- Use the website prompts to polish hero, cards, toolkit, AI Lab, changelog, and footer.
- Add the CSS tokens from `02-assets/css/linacre-brand-tokens.css`.
- Make sure the brand feels premium, not cluttered.

### Phase 3 — Build social consistency
- Use the social profile plan.
- Create avatar, header, post background, carousel template, and short video intro.
- Use the content calendar for repeatable posting.

### Phase 4 — Add audio identity
- Generate the 3-second sting, 8-second intro, 15-second social cue, and 30-second website loop.
- Keep audio subtle. The sound should feel like a signature, not background noise.

## Core Creative Rule

Linacre.site should feel like a **quiet workshop with amber screens, clean code, and a soft retro audio pulse**.

Avoid anything that feels too loud, too corporate, too gamer, too neon, or too generic AI-startup.
