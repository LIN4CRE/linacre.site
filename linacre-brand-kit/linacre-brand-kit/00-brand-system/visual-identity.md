# Visual Identity — Linacre.site

## 1. Visual Direction

The visual identity is **soft retro cyber with modern full-stack polish**.

It should feel like a late-night engineering dashboard: dark surfaces, amber signal light, clean card layouts, subtle terminal details, warm gradients, and hexagonal/pulse motifs.

### Core Visual Formula

```text
Deep ink background
+ warm amber accent
+ hexagon geometry
+ subtle pulse/motion
+ clean modern type
+ small terminal/code details
= Linacre.site
```

## 2. Colour Palette

### Primary Colours

| Token | Hex | Use |
|---|---:|---|
| Ink Black | `#070A0F` | Main dark background |
| Deep Slate | `#0B1220` | Cards, sections, elevated surfaces |
| Panel Slate | `#111827` | Card interiors, panels, footer |
| Amber Core | `#F59E0B` | Main brand accent, logo glow, buttons |
| Amber Glow | `#FFB000` | Highlights, hover states, subtle light |
| Warm Cream | `#F5E7C8` | Premium light text/accent on dark |

### Secondary Colours

| Token | Hex | Use |
|---|---:|---|
| Soft Cyan | `#22D3EE` | Rare tech signal accent |
| Muted Violet | `#A78BFA` | AI/lab accents, used sparingly |
| Error Coral | `#FB7185` | Warnings/errors only |
| Success Green | `#34D399` | Positive status only |

### Rule

Amber should do 80% of the brand work. Cyan/violet should only appear as small “AI signal” accents.

## 3. Typography

### Recommended Stack

- **Headings:** Space Grotesk, Sora, or Inter Tight
- **Body:** Inter, system-ui, Segoe UI, Arial
- **Code/Terminal:** JetBrains Mono, IBM Plex Mono, Fira Code, monospace

### Style

- Headings: confident, compact, not too decorative.
- Body: clean and readable.
- Code: used as a design accent, not as a wall of text.

### Example Type Hierarchy

```css
H1: 56–72px desktop, 40–48px mobile, letter-spacing -0.04em
H2: 36–48px desktop, 30–36px mobile
Body: 16–18px, line-height 1.65
Small/meta: 12–14px, uppercase or mono where useful
```

## 4. Logo System

### Recommended Logo Directions

#### Direction A — Hex Pulse Mark
A simple hexagon outline with a small amber pulse line or node inside. Best for favicon, app icon, social avatar, and loading states.

#### Direction B — Terminal Monogram
A stylised `L` mixed with a terminal cursor or command prompt symbol. Best for developer identity.

#### Direction C — Amber Node Network
A minimal connected-node mark inside a hexagon. Best for AI/full-stack identity.

### Logo Rules

- Keep it simple at favicon size.
- Avoid tiny text inside the icon.
- Use amber on dark as the primary expression.
- Create a monochrome version for flexibility.
- The mark should still work as a black/white silhouette.

## 5. Layout Style

Use:

- large spacing,
- rounded cards,
- thin amber borders,
- subtle grid backgrounds,
- soft radial glows,
- mono labels for metadata,
- command-line details as microcopy.

Avoid:

- overcrowded dashboards,
- heavy borders everywhere,
- too many gradients,
- chaotic animations,
- unreadable low-contrast text.

## 6. Imagery Style

### Use

- abstract amber grids,
- glassy terminal panels,
- macro shots of keyboards/screens if photographic,
- dark workbench setups,
- geometric backgrounds,
- soft retro synth-light effects,
- subtle code screenshots.

### Avoid

- generic robot faces,
- blue hologram clichés,
- overused stock “AI brain” images,
- crypto-style neon overload,
- fake 3D mascots unless intentionally minimal.

## 7. Motion Style

Motion should be a **pulse**, not a performance.

Use:

- slow amber breathing glow,
- hover lifts of 2–4px,
- soft border sweep,
- tiny terminal cursor blink,
- subtle card reveal,
- hexagon line draw,
- audio waveform pulse.

Avoid:

- fast spinning objects,
- aggressive glitching,
- large parallax that hurts readability,
- constant animation on all elements.

## 8. Audio-Visual Connection

The music identity and visual identity should match:

- Piano intro = minimal hero state.
- Warm Rhodes verse = toolkit/card browsing.
- Tiny chorus burst = launch/reveal moments.
- Digital pulse = logo animation and hover details.

This is how the brand becomes memorable across website and socials.
