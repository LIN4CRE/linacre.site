# Assets Folder — Linacre.site

Use this folder to store generated and exported brand assets.

## Recommended Subfolders

```text
02-assets/
├── css/      Brand CSS variables and web tokens
├── json/     Design token JSON
├── svg/      Editable/vector assets
├── logo/     Final logo exports and favicon package
├── audio/    Stings, loops, social music beds
├── video/    Logo animations, video intros, reels backgrounds
├── social/   Banners, avatars, post templates, carousel templates
└── web/      Open Graph images, hero backgrounds, UI textures
```

## Naming Rule

Use this pattern:

```text
linacre-[asset-type]-[description]-[size-or-duration]-v01.[extension]
```

Examples:

```text
linacre-logo-hex-pulse-dark-v01.svg
linacre-social-linkedin-banner-1584x396-v01.png
linacre-audio-sting-03s-v01.wav
linacre-video-logo-sting-03s-v01.mp4
linacre-web-og-card-1200x630-v01.png
```

## Export Quality

- SVG for logos and icons wherever possible.
- PNG for social and web images.
- WAV master + MP3 web version for audio.
- MP4 H.264 for social video.
- Keep source/editable files when possible.

## Starter Assets

The `svg/` folder contains simple starter SVGs generated from the brand direction:

- `linacre-mark.svg`
- `linacre-wordmark.svg`
- `social-avatar.svg`

These are not a final professional logo package, but they give you a clean starting point and visual reference.
