### 1. About Page — Expanded Version

**Current (abbreviated):**
> I'm a full-stack engineer and digital operations architect focused on constructing robust, secure, and beautiful developer utilities, automation systems, and AI sandboxes.

**Recommended rewrite:**

```
# About David Linacre

## The short version

I build reliable software for startups that need to ship fast. 
React, TypeScript, Go, Python — whatever the stack demands. 
Systems audits from £1,800, custom builds from £6,500.

## How I got here

I started coding because I was frustrated with bloated, slow tools 
that got in the way instead of helping. That frustration turned into 
a career building the opposite: clean, fast systems that do one thing well.

Over the past several years I've shipped 17+ production systems — from 
high-throughput email engines in Go to full-stack React dashboards to 
AI-powered developer tooling. My code runs on Vercel, in Docker containers, 
and in browser sandboxes.

## How I work

- **One source of truth.** No duplicated data, no clutter.
- **Ship early, iterate fast.** You'll see working code in days, not months.
- **NDA-friendly by default.** Your code stays yours. Always.
- **UK-based, remote-first.** Async updates, scheduled calls, replies within 12 hours.

## When I'm not coding

I live in West Yorkshire, England. When I'm not at the keyboard, 
you'll find me tinkering with home automation, reading sci-fi, 
or walking in the Dales.

## Let's talk

If you need an engineer who treats your product like their own, 
[start a conversation →](/contact)
```
